# Employee – Java

Exemplo simples em Java para demonstrar Programação Orientada a Objetos.

## O que o projeto mostra
- Composição de classes (`Employee` usa `Date`)
- Construtor com parâmetros
- Sobrescrita do método `toString()`
- Classe de teste (`EmployeeTest`)

## Execução

javac date/*.java
java date.EmployeeTest
